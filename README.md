# Sistema de Cálculo de Lucros Cessantes

Autor: Anderson Torres  
Contador Público e Perito Contábil  

---

## Descrição

Este projeto foi desenvolvido em Python com o objetivo de automatizar o cálculo de lucros cessantes, utilizando diferentes cenários de margem de lucro.

O sistema permite calcular com precisão os valores indenizatórios com base em parâmetros utilizados na prática pericial contábil.

---

## Funcionalidades

- Cálculo automático de lucros cessantes
- Simulação com múltiplas margens de lucro:
  - 20%
  - 25%
  - 30%
  - 35%
- Entrada personalizada de:
  - faturamento mensal
  - quantidade de meses
- Exibição clara dos resultados

---

## Aplicação na Perícia Contábil

Este sistema pode ser utilizado em:

- Perícia judicial
- Perícia extrajudicial
- Cálculo de indenizações
- Apuração de lucros cessantes
- Análises financeiras
